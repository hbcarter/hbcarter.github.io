POSTER PROJECT

The poster project taught the process of creating hierarchy with positioning elements on a page ,such as, text and visual images to guide the viewers perception of what should be read first to last. In this project we manipulated scale, weight, and line in order to guide the readers eyes to set up a clear hierarchy to prevent confusion for the viewers.The color study showed how adding color to the poster project creates different dynamics and moods depending on the color combinations being put into use. This process shows how to implement these various color theory techniques that we learned in Design Theory to convey an overall message.
